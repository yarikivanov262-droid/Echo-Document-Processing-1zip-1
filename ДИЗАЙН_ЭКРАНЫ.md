# ECHO — редизайн по экранам (ориентир: MAX / web.max.ru)

Этот документ дополняет `ДИЗАЙН_СИСТЕМА.md` конкретными указаниями по каждому экрану. Используется как техзадание для дизайнера/разработчика при доработке интерфейса.

---

## 1. Экран входа / регистрации (`src/pages/auth/login.tsx`)

**Сейчас:** тёмный фон с размытыми голубыми/фиолетовыми пятнами, карточка по центру, весь текст на английском, моноспейс-шрифт, uppercase-кнопки с трекингом букв — выглядит как экран входа в криптокошелёк, а не в мессенджер.

**Должно стать:**
- Логотип ECHO — крупный, простой значок (щит уже используется — оставить, но убрать неоновое свечение, сделать плоским, в тёплом акцентном цвете).
- Заголовок «ECHO» и слоган «Говори. Никто не узнает.» — оставить как есть, это хорошо работает.
- Поля ввода — без моноспейса, обычный шрифт, скругление 14px, заливка `--card`, без иконки-рамки в стиле терминала.
- Кнопка входа — «Войти», заливка градиентом, без uppercase.
- Ссылка «Создать аккаунт» — обычным текстом акцентного цвета, без uppercase.
- Экран регистрации в 3 шага (сейчас реализован верно по структуре — оставить):
  1. Пояснение, что не нужен телефon/email → кнопка «Создать кодовую фразу».
  2. Показ 12 слов сеткой 3×4 (сейчас так и есть — хорошо) → кнопки «Обновить» / «Скопировать» → «Я сохранил(а) фразу».
  3. Ввод имени пользователя → «Готово».
- Убрать все `uppercase tracking-widest` — заменить на обычный текст с нормальным начертанием (semibold для кнопок).
- Полностью перевести интерфейс регистрации на русский (сейчас смешение языков).

## 2. Список чатов (`src/pages/chat/chat-list.tsx`)

**Сейчас:** структура close к MAX (шапка «Чаты», кнопка добавить, вкладки, список), но цветовая палитра и форма карточек — с угловатыми элементами и голубым акцентом.

**Должно стать (по образцу скринов MAX):**
- Шапка: заголовок «Чаты» слева, две иконки справа (меню «•••» и «+» в акцентном круге) — уже так, только перекрасить «+» в тёплый градиент.
- Вкладки-фильтры («Все», «Новые», папки) — активная вкладка с подчёркиванием акцентным цветом снизу, неактивные — серым текстом. Бейдж счётчика рядом с названием вкладки — маленький серый кружок с числом (не путать с бейджем непрочитанных в самом чате).
- Поиск — строка поиска с лупой над списком чатов (сейчас есть отдельная страница поиска — рассмотреть встраивание строки поиска прямо в шапку списка, как на скрине MAX, вместо отдельного перехода).
- Элемент чата: аватар слева (56px, круглый, градиентная заливка если нет фото), имя + последнее сообщение, время и бейдж непрочитанных справа. Бейдж — заливка акцентным красным цветом, а не голубым/серым.
- Свайп/долгое нажатие по чату → контекстное меню: «Добавить в папку», «Открепить/Закрепить», «Отметить непрочитанным», «Отключить уведомления», «Удалить чат» — набор пунктов уже соответствует MAX, нужно только привести иконки к line-style и перекрасить деструктивный пункт в красный.

## 3. Окно переписки (`src/pages/chat/chat-window.tsx`)

**Сейчас:** пузыри сообщений одинаковой формы, нет визуальной группировки подряд идущих сообщений, светлый акцент на исходящих.

**Должно стать:**
- Исходящие сообщения — градиентная заливка (тёплый акцент), выравнивание вправо.
- Входящие — заливка `--card`, выравнивание влево.
- Группировка: если несколько сообщений подряд от одного отправителя в течение нескольких минут — показывать аватар/имя только у первого, скруглять пузыри «внутри группы» сильнее с одной стороны, оставляя выраженный хвостик только у последнего.
- Пустое состояние чата («Сообщений пока нет») — как на скрине MAX: карточка по центру с текстом и иллюстрацией/стикером. Сейчас, вероятно, просто пусто — добавить дружелюбное пустое состояние.
- Поле ввода снизу: скрепка слева, поле «Сообщение» по центру, дальше — эмодзи/стикеры и кнопка голосового сообщения справа (сейчас структура похожая — проверить, что иконка камеры/скрепки визуально соответствует line-style).
- Панель эмодзи/стикеров снизу — вкладки «Стикеры» / «Эмодзи» внизу панели, категории иконками сверху, сетка стикеров/эмодзи — уже есть похожая структура в проекте, нужно применить новую палитру (тёмный фон панели `--card-elevated`, активная категория подсвечена акцентом).
- Долгое нажатие на сообщение → всплывающее меню реакций (эмодзи-панель сверху) + меню действий (Ответить, Переслать, Отметить непрочитанным, Скопировать текст, Удалить, Выбрать) — состав пунктов уже соответствует MAX, актуализировать только визуальный стиль (тёмная карточка, красный цвет только у «Удалить»).

## 5. Настройки — главный экран (`src/pages/settings/settings.tsx`)

**Должно стать (по образцу скрина MAX «Настройки»):**
- Шапка профиля сверху: аватар, имя, идентификатор — карточкой, с иконкой редактирования.
- Блок быстрых действий: «Вернуть уведомления» / «Пригласить друзей» — карточками с цветной иконкой слева (уже частично реализовано похоже, актуализировать цвета иконок под новую палитру: акцент + фиолетовый для второстепенных).
- Список разделов карточками с разделителями: «Уведомления и звук», «Безопасность», «Устройства», «Сообщения», и т.д. — иконка слева в цветном квадрате-бейдже (16-20px, скругление 8px), название по центру, шеврон справа. Эта структура уже подходит, нужно только привести цвета бейджей иконок к единой системе (каждый раздел — свой акцентный цвет иконки, как на скрине MAX: жёлтый для уведомлений, синий/фиолетовый для безопасности и т.д., на тёмных карточках-квадратах).

## 9. Общие правки, применимые ко всем экранам

- Заменить голубой акцент на тёплый красно-коралловый везде (кнопки, активные вкладки, тумблеры, ссылки).
- Убрать `uppercase` и `tracking-widest` с текста кнопок — использовать обычный регистр.
- Увеличить радиус скругления карточек/полей/кнопок с 12px до ~16-18px.
- Все деструктивные действия (удалить, заблокировать, выйти) — красным текстом, без заливки фона, по образцу MAX.
- Иконки — везде line-style (обводка), одинаковая толщина линии, без смешения filled/outline иконок на одном экране.
